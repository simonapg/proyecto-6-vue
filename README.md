# Tiempo en Chile

Aplicacion SPA desarrollada con Vue 3, Vue Router y SASS para consultar el clima de distintas localidades de Chile.

La app permite visualizar datos climaticos actuales, filtrar lugares de forma reactiva y navegar a una vista de detalle con pronostico semanal, estadisticas y cambio inmediato de unidad de temperatura. La informacion meteorologica se obtiene desde la API de Open-Meteo.

## Estructura

El proyecto principal se encuentra dentro de la carpeta `vue-project`.

## Tecnologias

- Vue 3
- Vue Router
- Vite
- SASS
- Open-Meteo API

## Ejecucion

```sh
cd vue-project
npm install
npm run dev
```

## Build

```sh
cd vue-project
npm run build
```
